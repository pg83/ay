Key facts:

- The namespace KEY path is `mod_root_path` = the *resolved* root-relative path
  of the source (`rootrel_arc_src(path)`) with its trailing `/<token>` removed —
  NOT the module dir.
- For a module-local token `foo.py`, `rootrel_arc_src` = `<moduledir>/foo.py`, so
  `mod_root_path = <moduledir>` (current behavior).
- For an arcadia-root-relative token (e.g. codegen_bin's
  `PY_SRCS(yabs/.../bin/__main__.py)` where the file is checked in at that root
  path, not under the module), `rootrel_arc_src` returns the token itself, so
  `mod_root_path = ""` → key `py/namespace/<md5>/=...`. This is the REF shape.
- For a `SRCDIR`-redirected token, `rootrel_arc_src` = `<srcdir>/<token>`, so
  `mod_root_path = <srcdir>`. This SUBSUMES the existing SRCDIR special case.
- The namespace VALUE (`ns`) stays module-derived (module dotted path, or
  `NAMESPACE`/`TOP_LEVEL` override) — only the KEY path is source-root-derived.
- `is_extended_source_search_enabled` requires `is_arc_src` → generated `$(B)`
  sources contribute no namespace entry (T-73 preserved). They still fold into
  `mod_list_md5`.

## Our existing machinery

- `resolvePySrcRel(fs, srcDirs, modulePath, srcRel)` in `resource.go:396` already
  computes exactly `rootrel_arc_src(token)` (without `$(S)`): SRCDIR reverse
  search → root-relative fallback → module-dir default. This is the seam — reuse
  it, do not invent a parallel resolver.
- `emitPyNamespaceForGroup` in `emit_py_objcopy.go:484` currently hardcodes
  `keyPath = instance.Path.rel()` with a `len(d.srcDirs)>1` SRCDIR special case,
  producing a single kv. That special case is an approximation of the rootrel
  mechanism and is removed by this change.

## Change

In `emitPyNamespaceForGroup`:

1. For each arc (non-generated) `.py` source in the group, compute
   `resolvedRel := resolvePySrcRel(ctx.fs, d.srcDirs, instance.Path.rel(), srcRel)`
   and `modRoot := resolvedRel[: len(resolvedRel)-len(srcRel)-1]` (clamp <0 → "").
2. Collect distinct `modRoot` values; emit one kv per distinct root, sorted, each
   `py/namespace/<md5>/<modRoot>=<nsValue>` (cmd) and `..."<nsValue>"` (hash).
   A group's `ns` is constant, so each root maps to the single `nsValue`.
3. Delete the `keyPath = instance.Path.rel()` default and the
   `len(d.srcDirs)>1` SRCDIR branch — both subsumed by (1).

`mod_list_md5` (over module names) is unchanged.
