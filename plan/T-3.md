# T-3 — Model LJ_21_ARCHIVE raw compilation + archive wiring

## Upstream mechanism

`build/plugins/lj_archive.py::LJ_21_ARCHIVE(NAME Name LuaFiles...)`:

1. luas = every arg ending in `.lua` (the `NAME`/`Name` slot words are skipped — they
   do not end in `.lua`).
2. For each lua `a`: `s = a[:-3] + 'raw'` (`.lua` → `.raw`) and
   `unit.on_luajit_21_objdump(['OUT', s, a])`.
3. `unit.onarchive_by_keys(['DONTCOMPRESS','NAME','LuaScripts.inc','KEYS',':'.join(luas)] + objs)`
4. `unit.onarchive_by_keys(['DONTCOMPRESS','NAME','LuaSources.inc','KEYS',':'.join(luas)] + luas)`

`_LUAJIT_21_OBJDUMP` (ymake.core.conf:4521):
```
.CMD=${cwd:LUAJIT_21_PATH} ${tool:"contrib/libs/luajit_21/compiler"} -b -g ${input:Src} ${noauto;output:OUT} ${hide;kv:"p LJ"} ${hide;kv:"pc light-cyan"}
LUAJIT_21_PATH=${ARCADIA_ROOT}/contrib/libs/luajit_21
```
→ LJ node: cmd `[compilerBin, -b, -g, $(S)/<mod>/<lua>, $(B)/<mod>/<raw>]`,
cwd `$(S)/contrib/libs/luajit_21`, env `ARCADIA_ROOT_DISTBUILD=$(S)`,
inputs `[compilerBin, $(S)/<mod>/<lua>]`, tool dep on the luajit compiler,
kv p=LJ pc=light-cyan, requirements cpu1/net restricted/ram32, module_dir=<mod>.

`ARCHIVE_BY_KEYS` (ymake.core.conf:4174):
```
.CMD=$ARCH_TOOL -q -x $DONTCOMPRESS ${input:Files} -k $KEYS -o ${addincl;noauto;output:NAME} ...
```
Differs from `ARCHIVE` only by: plain `${input:Files}` (no `:` suffix) followed by
`-k <:joined keys>` before `-o NAME`.

Verified against `/home/pg/monorepo/4/sg7.json`: 356 LJ `.raw` nodes + `LuaScripts.inc`
(archives the 356 `.raw`) + `LuaSources.inc` (archives the 356 `.lua`), keys = the 356
module-relative lua names joined by `:`.
