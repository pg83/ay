## Cluster 1 — PCRE_JIT (22 L3, root cause identified)

All 22 nodes are `[CC] contrib/libs/pcre/*.c.o` missing `-DARCADIA_PCRE_ENABLE_JIT`.

**Upstream root cause** (`yatool_orig/contrib/libs/pcre/ya.make:33-39`):
```
DEFAULT(ARCADIA_PCRE_ENABLE_JIT yes)
IF (ARCADIA_PCRE_ENABLE_JIT)
    CFLAGS(-DARCADIA_PCRE_ENABLE_JIT)
ENDIF()
```

**OUR bug**: `gen.go:1070-1078` stores `DefaultVarStmt` into `d.defaultVars` for `$CFG_VARS` expansion only; it does NOT mutate the `env` used by `EvalCond` on subsequent `IF` statements in the same module. `macros.go:379` pins `ARCADIA_PCRE_ENABLE_JIT=false` in `DefaultIfEnv`, so the `IF` arm is skipped. Same pattern affects any module using `DEFAULT(X yes)` + `IF (X)` — search the closure for further latent cases.

**Fix sketch**: in the walker (gen.go:944 collectStmts), when a `DefaultVarStmt` is seen, clone the env and bind the variable so subsequent IfStmts in the same body see it. Module-scoped only — DEFAULT does not propagate across modules.

## Cluster 2 — PY/contrib/tools/python3 objcopy duplicate inputs (40 L2)

40 `[PY] contrib/tools/python3/Lib/objcopy_*.o` nodes have duplicated yapyc3 + .py pairs in `inputs[]`. Programmatic check: after deduping OUR inputs, **40/40 match REF exactly**.

**Root cause** (`resource.go:970-975` in `emitPySrcObjcopy`):

```go
inputs := []string{
    "$(BUILD_ROOT)/tools/rescompiler/rescompiler",
    "$(BUILD_ROOT)/tools/rescompressor/rescompressor",
}
inputs = append(inputs, ch.inps...)   // ← contains duplicates

```

The comment at line 964-969 documents intentional duplication "on chunk-straddles", but the in-chunk path also emits each entry twice (kv_only + path+key slots, both adding the same `.yapyc3` + `.py`). REF's single-chunk shape is deduped.

**Fix sketch**: `dedupPreserveOrder(ch.inps)` before append, OR fix `chunkPySrcEntries`/`buildPySrcEntries` so `ch.inps` is unique by construction. Verify on rapidjson (multi-chunk straddle case) that genuine cross-chunk duplication isn't lost.

**Cascade**: 40-node fan-in to AR `libpy3tools-python3-Lib.global.a`. Closing this collapses the single largest consumer hub.
